## NoTab 到底是个啥

简单说，它是一个浏览器扩展，Chrome、Edge、Firefox 都能装。核心就一句话：**在当前页面里直接预览链接，不用跳新标签**。

按住快捷键加点击，或者悬停触发（看你怎么设置），一个小窗口就弹出来，把目标页面的内容直接渲染给你看。看完关掉，主页面纹丝不动，刚才读到哪还在哪。

它的开发者把它定位成"链接预览助手"，但其实它不止预览——还顺手把翻译和搜索也塞进来了。选中文本，一键开小窗翻译或丢给多个搜索引擎查，不用再单独开 Google Translate 标签页来回贴。

## 我用它最多的三个场景

**第一个，刷信息流。** Reddit、V2EX、Hacker News 这种聚合站，标题党满天飞。以前点进去发现是水文，关掉，再点下一个，又关掉，标签页开开关关像在打碟。现在悬停预览一眼，值不值得点进去一目了然，省心。

**第二个，看 YouTube。** 它有个视频模式，能把页面里的杂七杂八元素扒掉，只留视频和正文。学习类长视频开沉浸模式看，注意力不再被右侧推荐栏的"下一个"勾走。

**第三个，读外文文档。** GitHub 的英文 README、海外博客的技术文章，选中段落直接小窗翻译，原文译文并排看，比开两个标签页来回切舒服多了。
